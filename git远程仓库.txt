0. 创建本地仓库
   创建一个文件夹 tutorials
   进入文件夹  
   git init

.1 查看工作树和索引状态
   git status

.2 往索引添加文件(多文件，空格分割)
   git add 文件全名(路径)
   git add .             //把所有文件加入到索引 

.3 添加后需执行commit命令
   git commit -m "提交备注(自己写,必须要有)"
   gitk   打开查看工具GUI

1. 添加远程仓库
   git remote add [仓库名]  https://github.com/mchjmwjw/FirstRep.git

2. 查看当前远程库
   git remote

3. 推送远程仓库
   git push -u [仓库名:origin] [分支:master]

4. 删除远程仓库
   git remote rm [仓库别名]
